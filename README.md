# Errday

All day. Errday.

Errday is a private personal operating system for fitness, nutrition, sleep, journaling, body weight tracking and daily habits.

## Core modules

- Today dashboard
- Gym tracker
- Food and calories
- Sleep tracking
- Journal
- Body weight and progress
- Future integrations: Strava, Apple Health, Health Connect

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- PWA-first mobile experience
